오류 수정 및 추가 기능 구현 요청

1. 오류:
- 현재 상태 터치스루 모드 alpha= 0.5, 표시 설정 투명도 100%. 이떄, 기본적으로 오버레이를 키면 터치스루 상태의 투명도가 적용됌. 터치스루 모드를 한번 껐다 켜야 표시 설정 투명도로 반영됌

2. 추가기능=편집모드 구체 기능:
- 캐릭터 고정 버튼: 
   - 캐릭터 고정 버튼을 on하면 현재 화면 상태 그대로 캐릭터가 고정되고, 파란색 테두리의 투명상자만 분리되어 움직인다. 
   - 이때, 투명상자를 클릭시 클릭 상태가 되며 꼭지점 또는 모서리를 드래그하여 그대로 크기를 변경할 수 있다.
   - 클릭 상태는 투명상자 외의 공간을 터치 시 풀린다.
   - 캐릭터 고정 버튼을 off하면 현재 캐릭터-투명상자간의 상대적 위치를 그대로 저장한다. 이는 표시 설정 크기 변경 및 이동 시 적용된다.
- 캐릭터 크기 변경 바:
   - 이 기능은 투명상자의 크기를 유지한채로 캐릭터의 크기만을 바꿀 수 있게 해준다. 이 상대적 크기는 저장되어 표시 설정 크기 변경 및 이동시 적용된다.
- 캐릭터 회전 입력 란: 투명상자와 분리하여 캐릭터의 회전을 360도로 줄 수 있다.정수입력
- 디스플레이 편집 프리셋 저장 버튼&보기 버튼:
   - 프리셋 저장은 현재의 모든 상대적 위치, 크기, 회전, 초기 위치(=현재 위치)을 저장한다. 이때, 이름을 설정할 수 있다.
   - 보기 버튼은 현재 저장된 프리셋을 팝업 메뉴를 띄워 보여준다. 해당 메뉴에서, 각 프리셋 옆에는 삭제, 불러오기, 링크 기능이 있다.
   - 링크 버튼 클릭시 현재 검색 가능한 live2d 모델을 팝업으로 보여주며, 선택 가능하다. 링크 기능은 live2d 모델 변경 시 적용된되며, 변경시 링크된 프리셋으로 불러와진다. 폴더 이름-프리셋 이름 기반으로 링크 상태가 저장된다.
