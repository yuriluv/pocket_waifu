# 오류 및 피드백
## 전체 기능 on/off(작동 안함)
- 기능 상세: 현재 알림과 live2d오버레이를 총괄하는 on/off. 이게 off면 둘의 세팅과 상관없이 강제로 off로 간주 된다.

## 알림(작동 여부 확인 불가)
- 지속 알림(영구적 알림) 기능 제거(오류 원인) 바람
- 알림은 선응답과만 연결되는 기능으로 고립. 단, 기존 구조 유지


## 선응답(작동 여부 확인 불가)
Looking up a deactivated widget's ancestor is unsafe.

At this point the state of the widget's element tree is no longer stable.

To safely refer to a widget's ancestor in its dispose() method, save a reference to the ancestor by calling

dependOnInheritedWidgetOfExactType() in the widget's didChangeDependencies() method.

이 로그가 뜸. 선응답 설정 방식을 엄밀하게 재정의하겠음. 
base=30m
deviation=10
overlayon=-20m
screenlandscape=+20m
screenoff=inf

합산 방식으로 변경. 이때, base에 음수 변수들을 더한 값이 음수이면, 오류.
inf는 해당 환경 변수에서 알림을 비활성화한다는 뜻.

# 미구현(미완성) 기능 피드백
## 1. live2d 기능 보완
현재 live2d는 오버레이와 배경투명화 정도만 오류 없이 작동함. 
이에 기존 live2d설정의 고급 설정과 상호작용 테스트 기능을 전부 제거하라.
다음은 제거 후 추가 내용이다:
- live2d-live2d 고급설정: 해당 공간에서 다음 기능을 조절한다. 각 탭마다 자동 모션, 제스쳐 시 모션 할당, 상호작용 테스트, 모션 파라미터. 
- 이때, 모션 파라미터에서 현재 할당 가능한 모션을 확인하거나, 지정할 수 있도록 세팅하라. 이에 대하여, 구체적인 기능을 상세히 갖춰야 하며, 사용자 친화적으로 세팅되어야 한다.

추가로 다음 기능도 제대로 작동하지 않는다:
- live2d 디스플레이 편집 모드: 편집 상태에서 지정한 데이터들을 전부 저장하고 불러올 수 있어야 하나, 저장-off-on-불러오기 시 제대로 작동하지 않음.
