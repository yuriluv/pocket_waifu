# 현재 목표
참조: 각 단계를 동시에 고려하면서 수행해야 문제가 안생길 듯 함.
1단계
- 배경 투명화 기능 = 흰색 상자(GLSurfaceView)는 투명하게. 이때 OpenGL (Renderer), Android (Kotlin) 모두 건들여야 함. 이때, 크기를 화면을 덮을 정도 또는 캐릭터(model mesh)보다 상대적으로 훨씬 크게 자동으로 설정한다. 이때, 흰색 상자는 히트박스 기능이 없어야 한다.
(부연: 크기를 과하게 키우는 이유는 커스텀 히트박스를 추가할 예정이고 live2d 캐릭터가 짤리지 않게 하기 위함)

2단계
- live2d 디스플레이 편집 모드 추가. 편집 모드는 추가로 다른 ui을 포함하지 않고, 작은 설정창을 하단에 띄우는 기능만 포함한다. 이때, live2d을 움직이게 할 수 있는 조정 가능한 사각형 또는 타원형 투명 히트박스를 추가하고 편집모드에서만 테두리가 보이게 설정한다. 

3단계
- 편집모드 설정에서는 live2d의 크기(바 형태로 조절), 회전 정도(360도), live2d의 위치 조정(설정 내의 화살표로 조정, 이때 live2d의 위치는 초기 오버레이 실행시 기본 위치로 저장된다.), 히트박스의 모양 및 크기 조절(가로 세로 각각 입력 가능한 정수로)를 가능하게 한다.
- live2d와 히트박스에 관계는 다음과 같다. 편집모드에서 지정한 live2d의 위치와 히트박스의 상대위치를 기억한다. 이에 따라 live2d가 히트박스에 실시간으로 동기화되어 따라간다.
