## main1. 팝업 미니 메뉴
알림의 메뉴 버튼을 눌렀을때 실행될 기능이다.
다른 앱 위에 그리기 기능을 활용하여, 팝업으로 미니 메뉴를 띄운다.

미니 메뉴는 상단에 탭이 3개 있으며,
각각 순서대로 일반, 입력, 설정 탭이다.

### 일반
일반 탭은 바로실행 가능한 버튼들로 구성되어있다.
- 스크린샷: 화면 공유 권한이 꺼져 있을 경우 우선, 권한을 요청한다. 그 외의 경우 **바로 팝업 메뉴를 종료 후**의 현재 화면을 캡쳐하여, llm에게 보낸다. 
   - 스크린샷 버튼 아래에는 입력창이 존재해야 한다. 만약 입력창에 메세지가 적여 있다면, 스크린샷을 보낼때 해당 메세지와 함께 보낸다. 메세지를 보낸 후 내용은 지워진다.
- 터치스루: 기존의 있는 기능인 터치스루 모드를 on/off 할 수 있는 토글로 지정한다.
- 앱으로 이동: pocket waifu앱을 즉시 시행한다.
- 알림 간편설정: 알림 기능을 on/off 할 수 있다.

### 입력
입력 기능은 알림의 답장 기능과 똑같이 동작한다.
여기서는 메인 채팅 세션과 비슷하게, 과거 채팅을 보여주고 입력창을 통해 메세지를 입력할 수 있게 하는 미니 채팅 화면이다.
여기서 입력 기능은 답장 기능과 연동된다.
