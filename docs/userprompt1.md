### 파라미터 실제 조절
조정 가능 파라미터를 전부 보여주고, 드래그로 조절 할 수 있게 한다. 이떄 파라미터 기능 이름은 영어 파라미터 변수 값 명을 그대로 표기한다.
이때 조절 값은 실제 live2d에 반영된다. 또한 상단에 reset 버튼을 추가하여 모든 파라미터를 초기화 할 수 있도록 한다.

### 명령어 입력
채팅 ui을 두고, 여기서 ai가 입력하는 xml명령어를 직접 쳐서 작동여부를 확인할 수 있다.(실시간)

### 모션 추가
상단에 api 프리셋을 선택할 수 있게 하고, 그 밑에 프롬프트 입력창을 두어, 해당 공간에서 해당 api로 채팅을 할 수 있게 한다. 이때, 이 api에 권한을 부여하여, json 파일을 수정, 추가할 수 있도록 두어라.
기본 프롬프트로는 영어로, 적당히 llm이 어떤 모션 프리셋을 만들어 json으로 두고, 이를 활용 가능 모션으로 저장하게 한다.
해당 공간은 단, 채팅 공간만 있으며, api 호출시 mcp 느낌으로 작동하는 특이사항만 있다.

## live2d llm 조절 기능 보완 및 구체화
live2d의 llm 조절은 다음 방식으로 실행되어야 한다.
api 호출 -> api 답변 -> response 단계에서 작동하는 lua 함수 수준에서 xml 명령어를 시행한다.-> 이후 해당 함수 수준에서 시행된 xml 명령어를 제거한다 -> 메인 채팅 세션에 필터된 답변이 보내진다. -> 알림 기능이 작동해야할 환경이면 알림으로 해당 답변이 전달된다.
